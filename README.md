# Roblox Offsets

A comprehensive collection of internal and external offsets for Roblox across Windows, Mac, and Android clients, maintained for the Roblox hacking community.

## 📋 Overview

This repository contains up-to-date offsets for Roblox across multiple platforms (Windows, Mac, and Android), including internal offsets, external offsets, and flags. These offsets are regularly updated to match the latest Roblox client releases.

## 🤝 Contributing

We welcome contributions! Here's how you can help:

* **Update Offsets:** Submit PRs with updated offsets for new Roblox versions
* **Add New Offsets:** Add new offsets you've discovered
* **Improve Documentation:** Help make the documentation clearer

### How to Update Offsets
1. Find the new offsets using your preferred method (IDA, Cheat Engine, etc.)
2. Update the corresponding `.hpp` files
3. Update the website HTML with the new values
4. Submit a PR with your changes
