This repository provides up-to-date Roblox client offsets and related data for reverse engineering purposes. Offsets are organized by platform and category, with raw files fetched directly from GitHub so you always get the latest version.

Use the tabs above to browse Windows, Mac, or Android offsets. Each platform contains internal offsets, external mappings, FFlags, configs, and structs where applicable. Click Copy to copy the raw file content to your clipboard.

How to Use

Select a platform tab above, then choose a category (Internal, External, FFlags, etc.) and pick a file. Click Copy to copy the exact raw content from GitHub. If an offset fails to resolve for Windows Internal, an alert will appear at the top of the viewer.
